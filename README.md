# 🔧 GitHub-OPS Skill

GitHub 仓库全生命周期运维管理技能。

## 覆盖范围

- 仓库审计
- 代码推送
- 分支管理
- PR 管理
- CI/CD 配置
- Issue 管理
- Release 管理
- 开发者引流
- 日常运维
- 错误处理

## 安装

```bash
# 克隆到 OpenClaw skills 目录
git clone https://github.com/xiejianjun000/github-ops.git ~/.openclaw/skills/github-ops
```

## 使用

设置环境变量：
```bash
export GITHUB_TOKEN="ghp_your_token"
```

然后让 AI Agent 使用 `github-ops` 技能。
